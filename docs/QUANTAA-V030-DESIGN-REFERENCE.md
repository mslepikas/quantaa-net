## Visual Direction

Build a professional real HTML/CSS website using the style guide:

- black / deep navy background
- neon blue and purple accents
- real Quantaa wordmark
- clean, professional header and footer
- real HTML text and navigation
- tasteful Q/circuit/star visual treatment based on the style guide
- services/technology/about/contact sections using published Quantaa.net content
